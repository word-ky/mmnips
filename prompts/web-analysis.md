# Web Analysis Prompt

Use this prompt with web GPT after refreshing the analysis pack:

```text
You are reviewing a local-to-AutoDL workflow. Read the attached files:

1. project-brief.md
2. latest-status.md
3. latest-log-tail.md
4. decision-log.md

Tasks:
- Summarize the current objective and system state.
- Identify the most likely issues or bottlenecks in priority order.
- Distinguish confirmed facts from inferences.
- Propose the next 3 validation steps.
- Propose the safest fix path and one fallback plan.
- Call out any operational or data-safety risks.

Respond in concise engineering language.
```
